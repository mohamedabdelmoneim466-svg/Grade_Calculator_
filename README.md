# Grade Calculator App 🎓

A simple C++ console application that takes a student's score and determines their grade level based on specific ranges. This is my **third project** in my C++ learning journey.

## 🚀 Features
* Validates user input (checks if the entry is a valid number).
* Categorizes grades into: Excellent, Very Good, Good, Pass, and Fail.
* Handles out-of-range numbers (0-100).
* Clean and organized console output.

## 🛠️ Built With
* **Language:** C++
* **IDE:** Visual Studio 2022

## 📝 How it Works
1. The program asks the user to enter a grade between 0 and 100.
2. It checks if the input is a number using `cin.fail()`.
3. It uses `if-else` statements to determine the grade:
   - **85 - 100:** Excellent
   - **75 - 84:** Very Good
   - **65 - 74:** Good
   - **50 - 64:** Pass
   - **0 - 49:** Fail
